What is the difference between map() and forEach() in JavaScript?

map() returns a new array with transformed elements, while forEach() executes a function for each element without returning anything.

What does the slice() method do to a string?

slice() extracts a section of a string and returns it as a new string without modifying the original string.

How can you check if an array includes a specific element?

Use the includes() method: array.includes(element) - returns true or false.
